# User Manual - Part 3 (Kampüs Yaşamı)

## Yemekhane
### Rezervasyon Yapma
1. Menüden yemekhane ve tarih seçin.
2. O günün menüsünü inceleyin.
3. "Rezervasyon Yap" butonuna tıklayın.
4. Bakiye kontrolü yapılacak ve rezervasyonunuz onaylanacaktır.

## Etkinlikler
### Etkinliğe Kayıt
1. Etkinlik takviminden ilginizi çeken etkinliği bulun.
2. Detaylarına tıklayıp kontenjan durumunu kontrol edin.
3. "Kayıt Ol" butonuna basın.
4. Ders programınızla çakışma yoksa kaydınız alınacaktır.

## Bakiye Yükleme
- Şimdilik öğrenci işleri üzerinden bakiye yüklemesi yapılmaktadır.
