# User Manual - Part 2 (Akademik & Yoklama)

## Öğretim Üyesi İçin
### Yoklama Başlatma
1. Dersinize girin ve "Yoklama Başlat" butonuna basın.
2. Sistem konumunuzu otomatik algılayacaktır.
3. Yoklama süresini ve izin verilen mesafeyi (ör: 50 metre) onaylayın.
4. Öğrencilere QR kodunu gösterin veya sistemi açmalarını söyleyin.

## Öğrenci İçin
### Yoklamaya Katılma
1. Dersteyken uygulamayı açın.
2. "Yoklamaya Katıl" butonuna tıklayın.
3. Konum izni verin.
4. Eğer sınıftaysanız (hoca ile aynı konumdaysanız) "Başarılı" mesajı alacaksınız.

## Notlar
- Konum servislerinizin (GPS) açık olduğundan emin olun.
- Bina içi çekim sorunlarında pencereye yaklaşmayı deneyin.
