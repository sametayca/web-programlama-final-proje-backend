# User Manual - Part 1 (Giriş ve Profil)

## 1. Sisteme Giriş
1. Uygulamayı açın.
2. "Email" ve "Şifre" alanlarını doldurun.
3. "Giriş Yap" butonuna tıklayın.
4. Başarılı giriş sonrası Ana Sayfaya yönlendirileceksiniz.

## 2. Kayıt Olma
1. Giriş ekranında "Kayıt Ol" linkine tıklayın.
2. Ad, Soyad, Email ve Şifre bilgilerinizi girin.
3. Email adresiniz kurumsal (@kampus.edu.tr) olmalıdır.
4. Kayıt tamamlandıktan sonra giriş yapabilirsiniz.

## 3. Profil Görüntüleme
1. Sağ üst köşedeki profil ikonuna tıklayın.
2. "Profil" seçeneğini seçin.
3. Kişisel bilgilerinizi buradan görüntüleyebilirsiniz.
