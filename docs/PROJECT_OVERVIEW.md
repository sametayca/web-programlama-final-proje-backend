# Akıllı Kampüs Yönetim Platformu - Project Overview

## 1. Proje Tanımı
Akıllı Kampüs Yönetim Platformu, üniversite kampüs yaşamını dijitalleştiren, öğrenci, akademisyen ve idari personel arasındaki etkileşimi artıran kapsamlı bir yönetim sistemidir.

## 2. Temel Amaçlar
- **Verimlilik:** Kampüs operasyonlarını otomatize etmek.
- **Erişilebilirlik:** Mobil uyumlu arayüzlerle her yerden erişim sağlamak.
- **Entegrasyon:** Farklı sistemleri (Yemekhane, Yoklama, Duyurular) tek çatı altında toplamak.

## 3. Teknoloji Yığını
- **Frontend:** React.js, Material UI
- **Backend:** Node.js, Express.js
- **Veritabanı:** PostgreSQL (Sequelize ORM)
- **Gerçek Zamanlı İletişim:** Socket.io

## 4. Part 1 Kapsamı (Temel Altyapı)
- Kullanıcı Kimlik Doğrulama (Auth)
- Rol Tabanlı Erişim Kontrolü (RBAC)
- Profil Yönetimi
- Temel Veritabanı Tasarımı
